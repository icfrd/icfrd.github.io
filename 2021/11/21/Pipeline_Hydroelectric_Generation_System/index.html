专利名称 / PATENT NAME: 输送管发电系统 / PIPELINE HYDROELECTRIC GENERATION SYSTEM

简介 / BRIEF

结构简单，一次过水，多次发电！

Simple Structure, One-time Water Flow, Many-times Hydroelectric Generation!

  • 每根输送发电管100带有发电机200；

Each hydroelectric generation pipe 100 has one generator 200;

  • 每隔设定落差设置一组发电机；

Set one hydroelectric generation pipe 100 every given drop;

  • 多根管子通过连接器300连接成密闭的长管；

Multiple pipes are connected into long airtight pipe through the connector 300;

  • 多个发电机200所发的电通过导线400并入电网；

The electricity generated by multiple generators 200 is connected to the grid through wires 400;

  • 当落差可观，相当于沿着一根长距离管道建设了多个发电站。

As the drop is impressive, it is equivalent to building multiple hydropower stations along a long-distance pipeline.

  • 可以在高海拔地区建立几十米高的矮坝，控制水流、调节流量。

The water flow can be controlled and the discharge can be regulated by building low dam with tens of meters height at high-altitude areas.
